# Smart Fertilizer AI  
## نظام التسميد الذكي (الخيار والبندورة)

نظام ذكي يساعد المزارعين على اتخاذ قرار التسميد الصحيح اعتمادًا على:
- نظام خبير (Rule-Based System)
- تعلم آلي مُراقَب (Supervised Machine Learning)
- تقييم حالة التربة باستخدام pH

---

## فكرة المشروع

يعاني العديد من المزارعين من الاستخدام غير الدقيق للأسمدة، مما يؤدي إلى:
- هدر الموارد
- ضعف امتصاص العناصر
- تراجع الإنتاج الزراعي

يقوم هذا النظام بتحليل قراءات:
- N (النيتروجين)
- P (الفوسفور)
- K (البوتاسيوم)
- pH التربة

ثم يقترح نوع السماد والكمية المناسبة بناءً على الاحتياج الحقيقي للتربة والمحصول.
